# Real-time Inventory Management System (PERN Stack)

Sistem manajemen inventaris modern yang memungkinkan pelacakan stok barang secara real-time. Dibangun menggunakan arsitektur **PERN** (PostgreSQL, Express, React, Node.js) dan **Socket.io** untuk sinkronisasi data instan antar pengguna.

## Fitur Utama
* **Real-time Dashboard**: Update stok otomatis tanpa perlu refresh halaman.
* **Inventory Tracking**: Manajemen SKU, kategori, dan jumlah stok.
* **Transaction Logs**: Riwayat barang masuk dan keluar.
* **Responsive UI**: Tampilan yang optimal di perangkat desktop maupun mobile.

## Tech Stack
| Layer | Teknologi |
| :--- | :--- |
| **Frontend** | React.js, Tailwind CSS, Axios, Socket.io-client |
| **Backend** | Node.js, Express.js, Socket.io |
| **Database** | PostgreSQL |
| **Containerization** | Docker & Docker Compose |
